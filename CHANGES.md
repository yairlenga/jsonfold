# Changes

## 0.2.0 (In progress)

### Added

* Grid alignment for arrays and objects with identical structure.
* New `classic` preset preserving traditional JSONFold formatting.
* Expanded unit tests for grid behavior and API coverage.

### Changed

* `default` and `high` presets now include grid alignment.
* Public APIs have been aligned across language implementations.
* Documentation and examples updated.

### Internal

* Improved frame buffering and delayed emission logic.
* Better handling of grid candidates and width limits.
* Additional regression and conformance tests.

See `releases/0.2.0.md` for detailed release notes.
