JSONFold Java API

JSONFold provides hybrid pretty/compact JSON formatting. It keeps JSON readable while reducing unnecessary vertical space in small arrays, objects, and scalar runs.

Choosing an API

  • JSONFold — format existing pretty-printed JSON text.
  • JacksonJSONFold — serialize Java objects with Jackson and format the result.
  • GsonJSONFold — serialize Java objects with Gson and format the result.
  • JSONFoldWriter — streaming filter for advanced integrations.

Modules

  • jsonfold-core — core text filtering API.
  • jsonfold-jackson — Jackson integration.
  • jsonfold-gson — Gson integration.
  • jsonfold-cli — command-line interface.
