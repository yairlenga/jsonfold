# Changes in JSONFold 0.2.0

## New Features

### Grid alignment

Added optional grid alignment for folded rows with identical structure.

Grid formatting aligns columns in arrays and objects to improve readability:

```json
[
  [    1,  20, "Red",     300 ],
  [ 4000,  50, "Yellow",    6 ],
  [   70, 800, "Green",  9000 ]
]
```

Object rows with identical keys are also supported:

```json
[
  { "id":   1, "name": "alpha",   "qty":   12 },
  { "id":  20, "name": "beta",    "qty": 3000 },
  { "id": 300, "name": "charlie", "qty":    4 }
]
```

Grid formatting:

* applies only to folded rows;
* requires identical row structure;
* preserves JSON semantics;
* respects the configured width limit;
* automatically disables itself when alignment is not possible.

### New preset: `classic`

Added the `classic` preset, which preserves the traditional JSONFold 0.1 formatting behavior without grid alignment.

## Preset changes

The default presets have been revised:

| Preset    | Description                                  |
| --------- | -------------------------------------------- |
| `off`     | Pass-through mode                            |
| `none`    | Pretty printing only                         |
| `low`     | Conservative compaction                      |
| `classic` | Traditional JSONFold formatting              |
| `default` | Balanced formatting with grid support        |
| `high`    | More aggressive formatting with grid support |
| `max`     | Maximum compaction                           |

Diagnostic presets remain available:

* `pack`
* `fold`
* `join`
* `grid`

## API improvements

Public APIs have been aligned across language implementations.

Core operations are now consistently exposed as:

* `config()`
* `format_json()`
* `write_json()`
* `create_writer()`
* `fold_text()`

Compatibility APIs (`dump()`, `dumps()`, `stringify()`, etc.) remain available.

## Testing

Expanded unit tests cover:

### Grid formatting

* array grids;
* object grids;
* mixed numeric and string columns;
* width overflow handling;
* minimum and maximum row limits;
* incompatible row structures;
* mismatched object keys;
* nested containers.

### Presets

Added coverage for:

* `classic`
* `default`
* `high`
* `grid`

### API

Added tests for:

* configuration overrides;
* width overrides;
* writer statistics;
* stream filtering;
* compatibility interfaces.

### Regression tests

All outputs are verified to round-trip back to the original JSON data.

## Internal improvements

* Improved frame buffering for grid candidates.
* Better handling of streaming and delayed emission.
* Reduced unnecessary buffering.
* Improved consistency between language implementations.
* Updated documentation and examples.

## Compatibility

JSONFold 0.2 implement consistent API across all implementations.

It's still 0.X - so minor changes to API are possible.
